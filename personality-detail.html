Abaye in the Talmud

I was interested to see how the spread of teachings for a specific personality in the talmud varied by Masechta. The idea was that this might give some insight into where this individual focused on more in his learning.

To start off I chose Abaye, a forth generation Amora (died 337 CE). Using the sefaria data set, I wrote a python script to count the number of times Abaye's name was mentioned in each Masechta. Then I adjusted this count based on the number of words in each masechta, so the size of the masechta didn't affect the measure of what Masehctas Abaye taught in most frequently.

Below is the resulting treemap. Future steps would be to repeat this for multiple individuals, to allow for comparison.
